# mouse-controller
control mouse using hand

## install & running
```
python -m venv env && \
source env/bin/activate && \
python install -r requirements.txt && \
python main.py
```

## things used
[mediapipe](https://developers.google.com/mediapipe/solutions/vision/hand_landmarker)
[pyautogui](pyautogui.readthedocs.io/)
